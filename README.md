WideWorldImporters SQL Data Analysis Project

 Overview
This project demonstrates SQL analysis on the WideWorldImporters sample database, progressing from beginner to advanced queries.

It covers:

- Data Exploration & Descriptive Analysis
- Diagnostic & Comparative Analysis
- Predictive, Profitability & Strategic Analysis

The queries explore revenue trends, customer behavior, product performance, profitability, operational efficiency, and strategic recommendations.

 Folder Structure
- SQL/ - Beginner, Intermediate, and Advanced queries
- Docs/ - Summary of findings and insights

Key Features
- Use of GROUP BY, SUM(), COUNT(), AVG()
- Window functions (ROW_NUMBER(), RANK(), LAG(), LEAD())
- CTEs, CASE WHEN, correlated subqueries
- KPI modeling, forecasting, and strategic insights
